Lección 4

单词和发音

  • 字母 c 在与 a/o/u 构成音节时，发 [k] 音；在 e/i 之前时，发 [θ] 音。
  • cama 床
  • cosa 事物
  • cuna 摇篮
  • cena 晚饭
  • cine 电影院
  • 字母 q 后必须加 u 再与 e/i 构成音节，发 [k]。
  • queso 奶酪
  • quizá 也许
  • 字母 z 在 a/o/u 之前都是发 [θ] 这个音。和字母 c 在 e/i 之前的发音是一样的
  • zapato 鞋
  • zona 地区
  • zumo （水果）汁
  • agua 水
  • lengua 舌头；语言
  • suave 柔软的，温柔的
  • fuego 火
  • nuevo 新的
  • neuve 九
  • mutuo 相互的
  • cuota 配额
  • vacuo 空的

Lección 5

单词和发音

  • 字母 g 与元音 a/o/u 或辅音 l/r 组合，发 [g] 或 [ɤ] 音。字母组合 gu 加 e/i 也是发这个音。
  • g 在停顿后的词首，或者词中，词组内的 n 之后发塞音[g],舌头的后部与软腭闭合，气流冲开阻碍，爆破而出，声带同时振动。
  • ga,gue,gui,go,gu 如果不在停顿后的词首，或者词中、词组内的 n 之后时，发 [ɤ]。发音时，舌后与软腭不完全闭合，而是留出缝隙让气流通过，声带同时振动。
  • gato 猫
  • guerra 战争
  • gafas 眼镜
  • lago 湖泊
  • pago 支付
  • agosto 八月
  • 在 gue,gui 的组合中，如果字母 u 上带有分音符号，就要发音
  • cigüete 白葡萄
  • 字母 g 在与 e/i 组合的时候，发 [x]。发音时，小舌向舌后下垂，两者之间形成缝隙让气流通过。声带不振动。
  • genio 天才
  • girasol 向日葵
  • j 与所有的元音组合时，都发 [x] 这个音。
  • jefe 上司
  • joven 年轻的，年轻人
  • jamón 火腿
  • peine 梳子
  • seis 六
  • reina 女王
  • oiga 表示惊叹的语气词
  • oír 听见
  • hoy 今天
  • Suiza 瑞士
  • ruido 噪声
